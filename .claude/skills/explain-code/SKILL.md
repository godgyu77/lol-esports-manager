# Explain Code Skill

> 코드를 이해하기 쉽게 설명하는 스킬.
> 출처: jtw explain-code 기반

---

## 언제 사용하나

- 코드의 구조와 동작을 이해해야 할 때
- 새 팀원 온보딩 시 코드 설명이 필요할 때
- 복잡한 로직의 흐름을 파악해야 할 때

---

## 설명 순서

### 1. 한 줄 요약

이 코드가 하는 일을 한 문장으로 설명한다.

### 2. 비유

일상적인 것에 비유해서 전체 구조를 설명한다.

### 3. ASCII 다이어그램

호출 흐름, 클래스 관계, 데이터 흐름을 시각화한다.

```
예시:
[Client] → [Controller] → [Service] → [Repository] → [DB]
                              ↓
                         [LLM Provider]
```

### 4. 핵심 코드 워크스루

단계별로 무슨 일이 일어나는지 설명한다.

### 5. 주의할 점

흔한 실수나 오해하기 쉬운 부분을 명시한다.

---

## 사용법

- 파일 경로가 주어지면 → 해당 파일을 읽고 설명
- 개념이 주어지면 → 프로젝트 코드에서 관련 부분을 찾아 설명
