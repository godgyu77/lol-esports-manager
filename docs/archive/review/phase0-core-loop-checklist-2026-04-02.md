# Phase 0 Core Loop Checklist

## New Career
- 감독 모드 선택 가능
- 선수 모드 신규 시작 차단
- 팀 선택 가능
- 초기 훈련/전술 세팅 잠금 확인

## Season Flow
- 하루 진행 가능
- 유저 경기일에 프리매치로 전환
- 경기 결과 반영
- 뉴스/인박스 갱신

## Save Flow
- 수동 저장 가능
- 자동 저장 갱신
- 다른 슬롯 불러오기 가능
- 잘못된 슬롯 로드 시 오류 노출

## Staff Flow
- 팀별 기본 스태프 배치
- 유저 팀 감독 처리
- 코치 제안/거절 정상 동작
