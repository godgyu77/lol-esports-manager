# Phase 0 EA MVP Scope

## Supported Core Loop
- 새 커리어 시작은 감독 모드 기준으로 지원한다.
- 팀 선택 후 초기 운영 세팅을 완료해야 하루 진행이 열린다.
- 하루 진행, 유저 경기 진입, 결과 반영, 뉴스/인박스 갱신이 핵심 루프로 묶인다.
- 세이브는 슬롯별 DB 파일에 저장되고, 불러오기는 해당 슬롯 상태를 그대로 복원한다.
- 스태프/코치 흐름은 유저 감독 세계관, 팀별 초기 배치, 코치 제안/거절까지 지원한다.

## Included Systems
- 저장/불러오기
- Day 진행과 유저 경기 중단
- 경기 결과 반영과 시즌 진행
- 뉴스/인박스 읽음 상태와 배지 동기화
- 훈련/전술 초기 세팅 잠금
- 스태프/코치 기본 고용 흐름

## Phase 0 Success Criteria
- 새 커리어 시작에서 저장/불러오기까지 핵심 루프가 끊기지 않는다.
- 유저 경기일에는 자동 시뮬레이션이 대신 진행하지 않는다.
- 손상된 세이브는 조용히 무시하지 않고 명확한 오류를 보여준다.
- 플레이어 모드는 메뉴에 보이지만 신규 시작은 차단된다.
