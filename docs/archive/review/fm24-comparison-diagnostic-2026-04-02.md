# FM24 대비 종합 비교 진단서 및 가격 벤치마크
> 작성일: 2026-04-02  
> 비교 기준: Football Manager 2024 PC  
> 대상: `lol-esports-manager` 현재 repo 상태

## 1. Executive Summary

이 프로젝트는 이미 `FM식 감독 시뮬레이션 구조`를 e스포츠 장르에 이식하려는 야심이 분명하고, 실제로도 시즌 운영, 훈련, 전술, 스태프, 이사회, 계약, 뉴스, 미디어, 라이브 매치, AI 보조 레이어까지 넓게 뼈대를 갖춘 상태다. 코드 구조와 기능 범위만 보면 단순 팬 프로젝트 수준을 넘었고, `LoL판 FM`이라는 포지셔닝은 충분히 성립한다.

다만 FM24와 정면 비교하면 현재 프로젝트는 아직 `깊이보다 범위가 앞선 상태`다. 즉, 기능 개수는 많지만 각 시스템의 상호작용 밀도, UI의 정보 위계, 반복 플레이를 지탱하는 콘텐츠 농도, 전체 밸런스 신뢰도는 FM24보다 얕다. 지금 단계는 “FM에 영감을 받은 유망한 관리자 시뮬레이션”으로는 설득력이 있지만, “FM급 완성작”으로 보기엔 아직 격차가 뚜렷하다.

핵심 총평은 아래와 같다.

| 항목 | 판단 |
|------|------|
| 가장 큰 강점 | LoL e스포츠 도메인에 맞춘 관리자 판타지, 시스템 폭, AI 보조 활용, 라이브 매치 연출 |
| 가장 큰 약점 | UX 정돈 부족, 시스템 깊이와 상호작용 밀도 부족, 장기 플레이용 콘텐츠 리듬 부족 |
| FM24와의 공통점 | 시즌 운영, 백룸 스태프, 전술/훈련, 보드/재정, 뉴스/미디어, 저장/로드 구조 |
| FM24와의 핵심 차이 | 의사결정의 촘촘함, UI의 압축 능력, 반복 플레이 내구성, 밸런스/완성도 신뢰 |
| 현재 제품 포지션 | 인디 프리미엄 하단 ~ 얼리액세스 상단 |

## 3. Detailed Comparison

### 3.1 핵심 게임 루프와 장기 구조

**공통점**
- 둘 다 “하루/주/시즌을 관리하며 결과를 축적하는 감독 시뮬레이션” 구조를 가진다.
- 시즌 진행, 일정 확인, 경기 준비, 경기 결과 반영, 오프시즌 정리라는 큰 루프는 유사하다.

**차이점**
- FM24는 매 경기 전후 의사결정이 매우 촘촘하고, 장기적으로는 선수단 개편, 전술 정체성, 유소년, 보드, 커리어 이동이 서로 엮여 리텐션을 만든다.
- 현재 프로젝트는 일정 진행 루프는 이미 존재하지만, 비경기일 리듬이 반복적으로 느껴질 위험이 있고 장기 누적 보상이 FM만큼 촘촘하진 않다.

**격차 원인**
- 시스템 간 상호작용이 아직 `넓게 분산`돼 있고, 하나의 선택이 여러 페이지에 파급되는 체감이 약하다.
- 내부 리뷰 기준으로도 비경기일 피로, 이벤트 밀도, 경제 압박 구조, 성장 변주 부족이 지적됐다.

**따라가야 할 것**
- 시즌 중 비경기일의 전략적 사건 밀도
- 장기 커리어에서의 축적형 보상과 위기 구조

**차별화 가능 포인트**
- 실제 LoL 패치 메타, 국제전, 스크림, 밴픽 감각을 FM보다 더 촘촘하게 넣는 방향

### 3.2 감독/단장 판타지와 의사결정 밀도

**공통점**
- 감독이 전술, 경기 준비, 선수 관리, 스태프 활용, 보드 대응을 주도하는 구조다.
- 사용자가 “내 선택이 팀의 정체성을 만든다”는 판타지를 느끼게 하려는 방향이 분명하다.

**차이점**
- FM24는 기자회견, 라커룸, 인간관계, 계약, 출전시간, 보드 약속, 스태프 권한 위임이 한 덩어리의 역할감으로 이어진다.
- 현재 프로젝트는 감독 겸 단장 역할을 명확히 두고 있지만, 그 역할이 만들어내는 사회적/정치적 압력은 아직 얕다.

**격차 원인**
- 의사결정 결과가 숫자 변화로는 반영돼도, 인물 간 감정선과 조직 내부 반응이 FM보다 덜 입체적이다.

**따라가야 할 것**
- 단기 선택이 장기 신뢰와 불만으로 이어지는 체인
- “내가 어떤 감독인가”를 축적하는 정체성 피드백

**차별화 가능 포인트**
- 감독과 코치의 역할 분담을 LoL팀 운영 방식에 맞춰 더 직접적으로 시각화

### 3.3 선수/스태프/팀 관리 깊이

**공통점**
- 선수 성장, 폼, 만족도, 부상, 성격, 코치/스태프, 멘토링, 아카데미를 다룬다.
- 최근 작업으로 스태프 선호 역할, 감독 출신 코치, 코치 추천 브리핑까지 들어가며 FM식 뼈대가 강화됐다.

**차이점**
- FM24는 개별 인물의 성격, 관계, 히에라키, 계약 기대, 역할 기대가 더 입체적으로 충돌한다.
- 현재 프로젝트는 시스템은 많지만 인물 간 긴장과 관계망이 아직 규칙 중심으로 보이는 구간이 많다.

**격차 원인**
- 개별 인물의 서사, 미묘한 욕구 차이, 장기 추적되는 감정 히스토리가 부족하다.

**따라가야 할 것**
- 출전 시간, 역할 기대, 관계 네트워크, 멘토링 후폭풍
- 스태프 조언과 실제 퍼포먼스 연결 고리 강화

**차별화 가능 포인트**
- 코치 추천, 상대 분석, 스크림 피드백을 LoL 팀 매니지먼트 감성으로 강화

### 3.4 전술/훈련/분석

**공통점**
- 전술 페이지, 훈련 페이지, 분석/리포트, 프리매치 준비, 코치 추천이 존재한다.
- 최근에는 초기 세팅 잠금과 코치 브리핑으로 `설정 후 진행` 구조가 FM에 더 가까워졌다.

**차이점**
- FM24는 훈련과 전술이 매우 정교한 피드백 루프를 갖고 있고, 세트피스처럼 보조 스태프가 실제 편의성까지 떠받친다.
- 현재 프로젝트는 추천과 설정은 생겼지만, 훈련/전술/분석이 시즌 중 장기적으로 얼마나 누적되는지 체감은 더 키워야 한다.

**격차 원인**
- 추천안 적용 이후 성과 추적과 회고 화면이 아직 약하다.
- 전술이 바뀌었을 때 경기 내용이 어떻게 달라졌는지 해석해주는 계층이 부족하다.

**따라가야 할 것**
- 전술 변경 전후 차이 설명
- 훈련 성과의 누적 추적
- 상대 분석이 실제 준비 행동으로 이어지는 UX

**차별화 가능 포인트**
- 밴픽, 패치 메타, 라인 상성, 스크림 데이터를 한 몸처럼 묶는 e스포츠형 분석 체계

### 3.5 경기 연출과 중계 체감

**공통점**
- 둘 다 “직접 플레이보다 감독 시점 관전”이 핵심이다.
- 경기 중 정보판, 상태 보드, 해설/코멘터리, 전술 수정 요소가 존재한다.

**차이점**
- FM24는 실제 축구 경기의 흐름과 하이라이트 편집 감각이 축적돼 있다.
- 현재 프로젝트는 Pixi.js 미니맵, LoL식 HUD, 밴픽/라이브 중계 감성이 좋고, 테마적으로는 오히려 신선하다.

**격차 원인**
- 전략 선택이 경기 서사와 설명에 얼마나 깊게 반영되는지는 아직 발전 여지가 있다.

**따라가야 할 것**
- 전술 선택과 경기 흐름의 인과 해설
- 경기 중 매니저 개입의 결과 가시화

**차별화 가능 포인트**
- LoL 중계 감성, 오브젝트 타이밍, 스노우볼 설명은 FM이 줄 수 없는 고유 장점

### 3.6 경제/이적/계약/보드

**공통점**
- 재정, 계약, 이적, 보드 요청, 스폰서, 시설 업그레이드가 모두 존재한다.

**차이점**
- FM24는 급여 구조, 계약 옵션, 에이전트/중개인, 보드 기대, 이적 협상 다층성이 매우 깊다.
- 현재 프로젝트는 시스템 분할은 좋지만 실제 압박감과 협상 드라마는 상대적으로 단순하다.

**격차 원인**
- 고정 지출 구조, 협상 긴장, 예산 제약, 실패 비용의 체감이 FM보다 약하다.
- 내부 리뷰에서도 경제 고정 지출 부족이 지적됐다.

**따라가야 할 것**
- 더 명확한 비용 구조
- 계약/이적 협상 선택지의 리스크
- 이사회 기대와 재정 제약의 실감

**차별화 가능 포인트**
- 스폰서/콘텐츠/IP형 수익 구조를 e스포츠 산업에 맞게 더 현대적으로 표현 가능

### 3.7 뉴스/미디어/서사

**공통점**
- 뉴스 피드, 인박스, 기자회견, 소셜 반응, 이벤트 체인이 존재한다.
- 최근 작업으로 코치 브리핑, 훈련 결과, 스크림 피드백 등 사소한 운영 정보도 더 잘 담기기 시작했다.

**차이점**
- FM24는 텍스트 완성도와 맥락 누적, 미디어 톤 일관성이 매우 높다.
- 현재 프로젝트는 AI와 템플릿 조합으로 양은 풍부하지만, 텍스트의 인물성·연쇄성·기억 효과는 더 필요하다.

**격차 원인**
- 실명/실명급 개별성, 사건 후속 기사, 내러티브 체인 밀도가 낮다.
- 내부 리뷰에서도 placeholder성 이름, 단조로운 기자회견 질문, 이벤트 텍스트 기계성이 지적됐다.

**따라가야 할 것**
- 기사 간 인과관계
- 선수/코치 개성 반영 텍스트
- 장기 시즌 톤 변주

**차별화 가능 포인트**
- AI를 이용한 적응형 미디어, 스크림/패치 기반 코치 브리핑은 FM보다 앞서갈 수 있다

### 3.8 UI

**공통점**
- 매니저 게임답게 정보량이 많은 대시보드형 구조를 지향한다.
- 주요 화면군이 기능별로 잘 분리돼 있다.

**차이점**
- FM24는 엄청난 정보량을 비교적 압축적으로 정리해 보여주는 데 강하다.
- 현재 프로젝트는 화면 수는 많지만, 정보 위계와 그룹핑, typography 스케일, 포커스 상태, 대비, hover 일관성이 아직 정리 중이다.

**격차 원인**
- 내부 리뷰에서 텍스트 대비, 포커스 표시, 정보 과밀, 버튼 상태 일관성 문제가 반복 지적됐다.

**따라가야 할 것**
- 정보 계층 단순화
- 위험/중요/참고 분리
- 화면별 핵심 CTA 1순위 명확화

**차별화 가능 포인트**
- LoL 브로드캐스트 미학을 매니저 UI에 더 강하게 녹이는 브랜드 방향

### 3.9 UX

**공통점**
- 사용자가 시즌을 전진시키며 여러 페이지를 순환하는 구조다.

**차이점**
- FM24는 진입 동선과 반복 루프가 매우 다듬어져 있고, 정보량이 많아도 “다음에 뭘 해야 하는지”가 비교적 명확하다.
- 현재 프로젝트는 DayView 중심 재정리가 이뤄지고 있지만, 여전히 일부 흐름은 모달/페이지 경합, 반복 클릭, 준비되지 않은 설정 진입에서 혼선이 생길 수 있다.

**격차 원인**
- 기능은 많지만 플레이 우선순위를 알려주는 안내 계층이 아직 고르지 않다.

**따라가야 할 것**
- 메인 루프 우선순위의 일관성
- 결과 요약 UX
- 반복 행동 최소화

**차별화 가능 포인트**
- 코치 추천 원클릭 적용, 운영 브리핑, LoL 전술 온보딩은 FM보다 더 친절한 입문 UX가 될 수 있다

### 3.10 시각 디자인/브랜딩

**공통점**
- FM에서 영감을 받은 실무형 매니저 UI 톤을 일부 계승한다.

**차이점**
- FM24는 수년간 축적된 브랜드 일관성과 레이아웃 규율이 강하다.
- 현재 프로젝트는 `fm-theme.css` 등 방향성은 좋지만 화면별 마감 편차가 남아 있다.

**격차 원인**

**따라가야 할 것**

- 모달/패널 규모 체계화

**차별화 가능 포인트**

### 3.11 콘텐츠 양과 반복 내성

**공통점**

**차이점**

- 현재 프로젝트는 e스포츠 도메인 특성상 기반 모수가 작고, 실제로도 반복 구간 콘텐츠 변주가 더 필요하다.

**격차 원인**

**차별화 가능 포인트**

**공통점**

**차이점**
- FM24는 상용 제품 기준의 품질 신뢰를 전제로 한다.
- 현재 프로젝트는 테스트와 구조는 빠르게 좋아지고 있지만, 실플레이 회차, 접근성, 보안, AI 안전성, 장기 밸런스 검증은 아직 더 필요하다.

**격차 원인**
- 내부 리뷰 기준으로 접근성, AI 보안, 일부 빌드/테스트/구조 리스크가 아직 완전히 정리되지 않았다.

**따라가야 할 것**
- 키보드 접근성, focus, contrast, reduced motion
- AI 비밀값 저장/호출 보안
- 장기 자동 시뮬레이션 QA

**차별화 가능 포인트**
- Tauri 기반 로컬 앱 + 로컬 AI 조합은 잘 다듬으면 FM과 다른 기술 매력이 된다

## 4. Gap Analysis

현재 FM24와의 격차는 크게 4종류다.

| 격차 유형 | 설명 | 현재 프로젝트 상태 |
|----------|------|-------------------|
| 구조 격차 | 시스템끼리 이어지는 밀도 차이 | 넓게 많지만 서로의 후폭풍이 약한 구간 존재 |
| UX 격차 | 정보 정리, 클릭 피로, 진입 동선 차이 | 주요 병목이 여전히 남아 있음 |
| 콘텐츠 격차 | 반복 플레이를 버티는 변주량 차이 | 시즌 반복에 필요한 사건 밀도가 부족 |
| 상용화 격차 | 접근성, QA, 완성도 신뢰 차이 | 기술적 잠재력은 높지만 상용 제품 신뢰는 더 쌓아야 함 |

현재 가장 위험한 오판은 `기능 수 = FM급 완성도`라고 착각하는 것이다. 실제로는 기능 수보다 `선택의 맥락`, `결과의 후폭풍`, `정보를 읽는 피로`, `반복 플레이에서의 변주`가 더 중요하다.

반대로 가장 큰 기회는 분명하다. FM은 축구에 최적화된 전통적 거대 시뮬레이션이고, 이 프로젝트는 `e스포츠 패치 메타`, `스크림`, `밴픽`, `AI 기반 브리핑`, `방송형 경기 연출`이라는 강한 차별 요소를 이미 갖고 있다. 즉 “FM을 그대로 복제”하는 것이 아니라, `FM급 운영 깊이 + e스포츠 고유 리듬`으로 가야 한다.

## 6. Priority Recommendations

### 즉시 개선

1. DayView, ManagerHome, News/Inbox, Training/Tactics 중심으로 `핵심 루프 UX`를 더 정리
2. 경제/계약/보드 시스템에서 `고정 지출과 압박감` 강화

4. 기사/브리핑/코치 조언의 텍스트 체인을 강화해 반복 구간 피로 감소

### 중기 개선

1. 훈련/전술/분석을 하나의 누적 피드백 루프로 묶기
2. 감독 정체성, 선수 불만, 스태프 조언, 팀 사기 사이의 연쇄 강화
3. 국제전, 패치 메타, 세대교체, 라이벌리 축을 장기 커리어 서사로 연결

### 장기 차별화

1. `FM 복제`보다 `LoL e스포츠 운영 시뮬레이션` 고유성 강화
2. AI 기반 스크림 리포트, 밴픽 코치 브리핑, 방송형 경기 해설을 핵심 차별점으로 고도화
3. 실제 패치와 메타 변화가 장기 플레이를 재구성하는 구조 강화

## 8. Conclusion

이 프로젝트는 `FM의 설계 철학을 LoL e스포츠 매니저 장르로 옮기려는 시도`로서 이미 상당한 구조를 확보했다. 현재 상태만으로도 매력은 분명하고, 특히 LoL 메타·스크림·밴픽·코치 브리핑·라이브 중계 연출은 FM24가 줄 수 없는 강력한 차별점이다.

하지만 FM24와 같은 가격대나 완성도 기대를 바로 맞추기엔 아직 `UX 정돈`, `시스템 깊이`, `반복 내성`, `접근성/QA 신뢰`가 부족하다. 따라서 지금 시점의 현실적인 결론은 다음과 같다.

- FM24와 `철학적 공통점`은 충분하다.
- FM24와 `상용 완성도`는 아직 격차가 있다.
- 현재 프로젝트는 `저가 인디`가 아니라 `야심 있는 인디 프리미엄 후보`다.
- 지금 팔면 `₩19,900 ~ ₩29,900`이 가장 현실적이다.
- 내가 지금 실제 구매자라면 `₩24,900`에는 구매 의향이 있다.

이후 UX, 경제 압박, 장기 콘텐츠, 접근성, QA가 정리되면 이 프로젝트는 단순한 “LoL 팬 게임”이 아니라 `e스포츠판 FM`으로 강하게 자리 잡을 잠재력이 충분하다.
