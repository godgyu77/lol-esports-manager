# FM24 비교 기반 개선 로드맵
> 작성일: 2026-04-02  
> 기준 문서: `fm24-comparison-diagnostic-2026-04-02.md`

## 1. Summary

이 로드맵의 목표는 현재 프로젝트를 `기능이 많은 프로토타입`에서 `반복 플레이가 가능한 e스포츠 매니저 게임`으로 끌어올리고, 이후 `얼리액세스 출시 -> 정식 출시`까지 이어지는 현실적인 개발 순서를 확정하는 것이다.

핵심 전략은 아래로 고정한다.

- FM24를 그대로 복제하지 않는다.
- `FM식 운영 깊이`는 참고하되, `LoL e스포츠 고유 리듬`을 중심 경쟁력으로 키운다.
- EA 이전에는 `기능 추가`보다 `핵심 루프 안정화와 UX 정리`가 우선이다.
- 가격 전략은 현재 평가 기준으로
  - `EA: ₩19,900 ~ ₩29,900`
  - `정식 출시: ₩29,900 ~ ₩39,800`
  로 둔다.

## 2. Roadmap Overview

| Phase | 기간 | 목표 | 출시/제품 의미 |
|------|------|------|----------------|
| Phase 0 | 즉시 ~ 2주 | 안정화와 MVP 재정의 | “무엇을 팔지”를 먼저 고정 |
| Phase 1 | 3 ~ 6주 | 핵심 루프 UX 정리 | 유저가 쉽게 굴릴 수 있는 상태 확보 |
| Phase 2 | 4 ~ 8주 | 시스템 깊이 강화 | FM식 관리 게임 체감 형성 |
| Phase 3 | 4 ~ 6주 | 콘텐츠와 차별화 강화 | “LoL e스포츠판 FM” 인상 확립 |
| Phase 4 | 2 ~ 4주 | EA 준비와 출시 게이트 | 얼리액세스 진입 조건 충족 |
| Phase 5 | EA 이후 3 ~ 6개월 | 정식 출시 전 확장 | 가격 인상과 정식 출시 정당화 |

## 3. Phase Plan

### Phase 0: 안정화와 MVP 재정의

**목표**
- “지금 있는 기능을 다 살리는 것”이 아니라 “출시 가능한 핵심 루프만 남기는 것”

**우선 작업**
- 빌드/테스트/보안 크리티컬 이슈 우선 해결
- 저장/로드, Day 진행, 경기 결과 반영, 뉴스/인박스, 훈련/전술 잠금, 스태프/코치 흐름을 핵심 루프로 고정
- 플레이어 모드, 과한 AI 의존 기능, 3D 확장, 장식성 시스템은 EA 범위 밖으로 밀어냄
- EA 범위 안/밖 기능을 문서로 명확히 분리

**완료 기준**
- 새 커리어 시작 -> 팀 선택 -> 초기 세팅 -> 시즌 진행 -> 경기 -> 저장/불러오기까지 끊김 없이 가능
- 크리티컬 오류와 데이터 손상 리스크 제거
- EA 범위 문서화 완료

**산출물**
- `EA MVP 범위 문서`
- `핵심 루프 체크리스트`
- `크리티컬 버그 정리표`

### Phase 1: 핵심 루프 UX 정리

**목표**
- FM 대비 가장 큰 약점인 `정보 위계`와 `진행 피로`를 먼저 줄임

**우선 작업**
- `DayView`, `ManagerHome`, `News/Inbox`, `Training`, `Tactics`를 메인 루프 중심 화면으로 재정렬
- 오늘 해야 할 일, 리스크, 다음 경기, 코치 조언을 한 화면에서 이해되게 정리
- 뉴스/인박스는 읽음 상태와 목록 유지 규칙을 일관화하고, 기사/브리핑/알림의 역할을 분리
- focus, contrast, keyboard access, hover/focus state, reduced motion 등 접근성 기본기 정리

**완료 기준**
- 사용자가 튜토리얼 없이도 `다음 행동`을 3초 안에 파악 가능
- 비경기일 반복 피로 감소
- 접근성 P1 이슈 해소

**산출물**
- `핵심 화면 UX 정리 문서`
- `접근성 수정 체크리스트`
- `메인 루프 QA 시나리오`

### Phase 2: 시스템 깊이 강화

**목표**
- FM과의 격차인 `선택의 후폭풍`과 `운영 압박감`을 올림

**우선 작업**
- 경제: 고정 지출, 예산 압박, 계약/이적 실패 비용, 보드 요구 체감 강화
- 선수/스태프: 역할 기대, 만족도, 코치 선호 역할, 감독 정체성, 팀 사기와의 연결 강화
- 전술/훈련/분석: 추천 적용 후 성과 추적, 변화 원인 설명, 상대 분석과 실제 경기 준비 연결
- 시즌 진행: 비경기일 사건 밀도와 장기 후폭풍 증가

**완료 기준**
- 전술/훈련/스태프 결정이 경기력, 만족도, 예산, 뉴스 반응에 다층적으로 반영
- 시즌 중 운영 판단이 숫자만이 아니라 상황 판단으로 느껴짐
- FM식 “관리 게임” 감각이 체감되기 시작함

**산출물**
- `경제/운영 밸런스 리포트`
- `전술/훈련 영향 리포트`
- `시스템 인과관계 표`

### Phase 3: 콘텐츠와 차별화 축 강화

**목표**
- FM을 따라잡는 대신 `e스포츠판 FM`으로 기억될 차별점 강화

**우선 작업**
- 패치 메타, 스크림, 밴픽, 코치 브리핑, 상대 리포트를 하나의 운영 체계로 묶기
- 뉴스/서사를 사건 체인형으로 강화
- 국제전, 라이벌리, 세대교체, 장기 커리어 변주 추가
- 경기 연출은 LoL 방송 감성, 해설, 오브젝트 타이밍, 전술 해석 중심으로 개선

**완료 기준**
- “축구 FM의 약식 복제”가 아니라 “LoL e스포츠 운영 게임”이라는 인상이 분명해짐
- 3 ~ 5시즌 이상 반복 플레이 가능한 변주 확보
- 코치/미디어/패치가 장기 플레이를 재구성함

**산출물**
- `콘텐츠 확장 로드맵`
- `장기 커리어 이벤트 표`
- `차별화 포인트 문서`

### Phase 4: EA 준비와 출시 게이트

**목표**
- 제품 신뢰와 가격 정당성을 확보한 뒤 얼리액세스 진입

**필수 게이트**
- 자동 테스트 + 수동 플레이 QA 시나리오 고정
- 최소 `1시즌 자동/반자동 시뮬레이션 100회` 기준 밸런스/오류 검증
- 저장/불러오기, 시즌 종료, 오프시즌, 다음 시즌 진입 안정성 검증
- 접근성/보안/성능 체크리스트 통과
- 가격/스토어 설명/로드맵 문서 준비

**EA 출시 조건**
- 핵심 매니저 루프가 끊기지 않음
- 저장/로드 신뢰 가능
- 시스템은 넓기만 한 것이 아니라 몇 개 축에서 깊이 체감이 생김
- 사용자가 다음 업데이트 기대 포인트를 명확히 이해할 수 있음

**산출물**
- `EA 출시 체크리스트`
- `릴리스 QA 문서`
- `스토어 설명/가격 전략 초안`

### Phase 5: 정식 출시 전 확장

**목표**
- 정식 출시 가격대를 정당화할 수 있는 깊이와 완성도 확보

**우선 작업**
- 장기 커리어 서사, 국제전, 팀 역사, 스태프/선수 관계망 심화
- 콘텐츠 반복 내성 확장
- UI 통일감과 브랜딩 마감 강화
- 추가 모드나 DLC 후보는 정식 출시 후순위로 검토

**정식 출시 조건**
- 장기 플레이에서 변주와 리듬 유지
- 시스템 간 인과가 분명함
- EA 피드백 반영 완료
- `₩29,900 ~ ₩39,800` 가격대에 대한 체감 설득력 확보

**산출물**
- `정식 출시 범위 문서`
- `EA 피드백 반영 리포트`
- `정식 출시 품질 게이트 문서`

## 4. Workstream Priorities

### A. 제품/범위
- EA 전까지는 `핵심 루프 안정화 > 기능 추가`
- 정식 출시 전까지는 `차별화 강화 > 모드 확장`
- 가격 인상은 `완성도 게이트 통과 후`에만 검토

### B. UX/UI
- DayView 중심 루프 재정리
- 뉴스/인박스/브리핑 정보 구조 정돈
- 접근성 기본기와 시각 위계 정리

### C. 시스템/밸런스
- 경제 압박, 계약/이적 후폭풍, 만족도/사기 연결 강화
- 훈련/전술/분석이 실제 경기 결과와 누적 피드백으로 이어지게 보강

### D. 콘텐츠/차별화
- 코치/미디어/스크림/패치/밴픽을 한 운영 체계로 통합
- 장기 시즌에서도 버티는 이벤트/내러티브 체인 확장

### E. 품질/출시 운영
- 자동 테스트 + 수동 회귀 시나리오 병행
- 저장/로드와 시즌 전환 신뢰 최우선
- EA 전에는 기능 홍보보다 품질 신뢰 확보가 우선

## 5. Release Gates and QA

### EA 게이트
- 새 게임, 팀 선택, 초기 세팅, 하루 진행, 경기, 저장/로드가 문제 없이 연결됨
- DayView 중심으로 오늘 할 일과 다음 행동이 즉시 이해됨
- 뉴스/인박스 읽음 상태와 알림 배지가 일관되게 동작함
- 경제/계약/스태프/전술 선택이 장기 결과로 연결됨
- 1시즌 자동/반자동 시뮬레이션 100회에서 치명 오류 없음

### 정식 출시 게이트
- 3시즌 이상 반복 플레이에서 변주 유지
- 저장/로드/시즌 종료/다음 시즌 진입 회귀 테스트 통과
- EA 피드백으로 지적된 UX/밸런스/품질 이슈 해결
- 가격대 상향에 대한 내부 설득력 확보

## 6. Pricing Strategy

| 단계 | 권장 가격 | 전제 |
|------|-----------|------|
| EA | ₩19,900 ~ ₩29,900 | 핵심 루프 안정화, UX 정리, 저장/로드 신뢰 |
| 정식 출시 | ₩29,900 ~ ₩39,800 | 시스템 깊이, 콘텐츠 반복 내성, 품질 신뢰 확보 |
| FM급 프리미엄 재평가 | 별도 검토 | 추가 시즌/콘텐츠/QA와 장기 완성도 축적 후 |

가격 전략 기준:

- 지금 단계에서는 `풀프라이스`보다 `인디 프리미엄 하단 ~ 얼리액세스 상단`이 맞다.
- 가격 인상은 기능 수가 아니라 `반복 플레이 가치`, `UI/UX 성숙도`, `품질 신뢰`를 기준으로 한다.

## 7. Conclusion

이 프로젝트의 가장 좋은 길은 `FM을 닮은 게임`이 되는 것이 아니라, `FM급 운영 재미를 가진 LoL e스포츠 매니저 게임`이 되는 것이다. 따라서 로드맵도 기능 확장 중심이 아니라, `핵심 루프 안정화 -> UX 정리 -> 시스템 깊이 강화 -> e스포츠 차별화 -> EA -> 정식 출시` 순서로 밀어야 한다.

현재 시점에서 가장 중요한 판단은 단순하다.

- 당장 더 넣는 것보다 먼저 `정리`해야 한다.
- EA 전에는 `넓이`보다 `끊기지 않는 루프`가 중요하다.
- 정식 출시 전에는 `기능 수`보다 `반복 내성`과 `완성도 신뢰`가 중요하다.
