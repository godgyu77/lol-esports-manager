# FM24 대비 종합 비교 진단 및 가격 벤치마크

작성일: 2026-04-06  
기준 빌드: `main` @ `372f20f` 이후 워킹트리 기준  
비교 기준: `Football Manager 2024`와 2026-04-06 시점의 공개 스토어 가격

## 1. Executive Summary

이 프로젝트는 더 이상 단순한 LoL 팀 운영 프로토타입이 아니다. 현재 코드베이스는 42개 엔진 디렉터리, 62개 마이그레이션, 53개 테스트 파일, 332개 통과 테스트를 가진 중대형 매니저 시뮬레이션으로 성장했다. 감독 루프, 선수/스태프 관리, 이적과 계약, 뉴스와 서사, 라이브 매치 HUD, AI 보조 계층까지 갖추고 있어 “LoL판 FM”이라는 포지셔닝은 충분히 설득력이 있다.

다만 FM24와 정면 비교하면 아직 완성도보다 범위가 앞선 상태다. 즉, 시스템 수는 많지만 FM24급의 장기 플레이 응집력, UI 정보 구조, 경제/협상 디테일, 시즌 반복 내러티브 밀도는 한 단계 아래다.

현재 판단은 다음과 같다.

- 제품 방향성: 강함
- 시스템 범위: 강함
- 장기 반복 플레이 응집력: 중간
- UX 정교함: 중간
- 프리미엄 가격 정당화 수준: 아직 FM급 아님

## 2. Methodology

이번 재진단은 아래 네 축으로 수행했다.

1. 로컬 코드와 문서 검토
2. 테스트 신호 확인
3. FM24 및 인접 장르 가격 벤치마크 조사
4. 세션에 노출된 전체 스킬 168개 relevance screening

직접 깊게 적용한 핵심 스킬 관점은 아래와 같다.

- `product-manager`: 제품 포지셔닝, 출시 단계, 가격 정당화 기준
- `product-trend-researcher`: 시장 대안과 가격대, 세그먼트 기준선
- `game-designer`: 코어 루프, 반복 플레이, 시스템 응집력
- `design-ux-researcher`: 화면 구조, 행동 유도, 반복 UX 마찰
- `engineering-code-reviewer`: 구조 신뢰도, 테스트 신호, 유지보수 리스크
- `narrative-designer`: 뉴스/미디어/감독 서사와 장기 몰입감
- `support-executive-summary-generator`: 의사결정형 보고서 구조 정리

전체 스킬별 검토 기록은 [skill-review-matrix-2026-04-06.md](./skill-review-matrix-2026-04-06.md)에 분리했다.

## 3. Current State Snapshot

### 강한 점

- 감독 모드 핵심 화면이 이미 폭넓다.
- 라이브 매치가 단순 결과 출력이 아니라 방송형 HUD와 전장 표현을 갖추기 시작했다.
- 뉴스, 라이벌리, 프랜차이즈 서사, 경쟁 정체성 같은 장기 몰입 장치가 붙고 있다.
- AI가 코어 룰을 대체하는 구조가 아니라 보조/연출 계층으로 배치되어 있다.
- 테스트 신호가 좋다.
  - 2026-04-06 실행 기준 `53`개 테스트 파일, `332`개 테스트 통과

### 약한 점

- 기능 수 대비 화면 정보 구조가 아직 빡빡하다.
- 경제/계약/이적 협상은 “있다” 수준을 넘어 “긴장감이 있다” 수준까지는 덜 올라왔다.
- FM24가 잘하는 장기 서사 축적, 사람 간 관계의 누적 감정, 반복 시즌 속 변주가 아직 얕다.
- EA 전 가격 저항선을 넘기려면 저장/로드와 장기 세이브 검증, UX 압축, 가이드 구조가 더 필요하다.

## 5. Detailed Diagnosis

### 5.1 Product Manager 관점

확인한 것:
- 지금 이 게임이 “무엇을 팔고 있는가”
- EA와 1.0에서 각각 어떤 약속을 할 수 있는가
- 가격을 올릴 근거가 기능 수인지, 반복 가치인지

판단:
- 이 게임의 실제 판매 포인트는 “FM24 복제”가 아니라 “LoL esports 운영 감각을 가진 감독 시뮬레이션”이다.
- 따라서 가격 전략도 AAA 축구 매니저 정면 대결이 아니라, 인디 프리미엄 관리 시뮬레이션 축에 맞춰야 한다.
- EA에서 가장 중요한 것은 기능 추가보다 루프 정리다.

### 5.2 Product Trend Researcher 관점

확인한 것:
- 현재 스토어에서 관리 시뮬레이션이 어느 가격대에 서는지
- FM24가 현재 판매 중인지
- 사용자가 돈을 내는 기준이 깊이인지, 연출인지, 라이선스인지

판단:
- FM24는 2026-04-06 기준 Steam에서 더 이상 판매되지 않는다.
- 실제 현재 프리미엄 기준선은 FM26, F1 Manager 2024, WE ARE FOOTBALL 2024 같은 타이틀이 맡고 있다.
- 이 프로젝트는 장르적 독창성은 강하지만, 아직 가격 상단을 정당화할 정도의 라이선스, 장기 내러티브 밀도, UI 마감은 부족하다.

### 5.3 Game Designer 관점

확인한 것:
- 순간 루프, 세션 루프, 장기 루프가 연결되는지
- 선택이 체감되는지
- 시스템 수가 의미 있는 결정을 늘리는지, 아니면 메뉴 수만 늘리는지

판단:
- 코어 루프는 성립한다.
- 다만 일부 시스템은 “존재”는 하지만, 서로를 증폭시키는 연결 고리가 약하다.
- 특히 비경기일 반복의 의미, 전술/훈련 선택의 장기 후폭풍, 경제 압박의 체감이 더 선명해져야 한다.

### 5.4 UX Researcher 관점

확인한 것:
- 사용자가 지금 당장 해야 할 다음 행동이 보이는지
- 정보가 계층적으로 정리되는지
- 반복 플레이에서 클릭 마찰이 누적되는지

판단:
- 최근 Manager loop 정리는 분명히 좋아졌다.
- 그래도 FM24처럼 “많지만 길을 잃지 않는 UI” 수준은 아니다.
- 특히 EA 전에는 DayView, Inbox, News, Finance, Training 계열의 우선순위 표지를 더 강하게 해야 한다.

### 5.5 Code Reviewer 관점

확인한 것:
- 구조가 넓어진 만큼 테스트와 유지보수성이 따라오는지
- 최신 패치가 임시 봉합인지, 구조 개선인지
- 출시 리스크가 코드 신뢰성에서 오는지 UX에서 오는지

판단:
- 최근 패치는 단순 기능 추가가 아니라 `dayAdvancerTasks`, `broadcast HUD`, `systemDepthEngine`, `releaseDepthEngine`처럼 구조 단위 개선이 많다.
- 테스트 통과 신호도 좋다.
- 현재 가장 큰 제품 리스크는 “코드 붕괴”보다 “상품 마감도와 플레이 응집력” 쪽이다.

### 5.6 Narrative Designer 관점

확인한 것:
- 뉴스와 서사가 실제 플레이 기억을 남기는지
- 감독/팀/선수의 개별성이 누적되는지
- 시즌을 넘길 이유가 이야기적으로 생기는지

판단:
- 최근 프랜차이즈 내러티브, 경쟁 정체성, 뉴스 태그 강화는 올바른 방향이다.
- 다만 FM24처럼 “내가 아는 선수와 관계가 시간이 지나며 변한다”는 기억성은 아직 약하다.
- 가격을 더 받으려면 단순 기사 생성량보다 “기억되는 인물과 서사 연결”이 필요하다.

## 7. Pricing Recommendation

### 권장 가격

- EA: `$14.99 ~ $19.99`
- 1.0 정식 출시: `$24.99 ~ $29.99`

### EA 가격을 이 구간으로 보는 이유

- 시스템 범위는 이미 저가 인디를 넘는다.
- 하지만 FM26, F1 Manager 2024, WE ARE FOOTBALL 2024처럼 즉시 프리미엄 상단을 받을 정도의 라이선스 파워와 완성도는 아니다.
- `Motorsport Manager`의 중단 가격대보다는 높은 잠재력이 있으나, 아직 `$29.99+`를 EA에서 바로 받기엔 UX/반복 구조가 덜 다듬어졌다.

### 1.0 가격 인상 조건

아래 조건이 충족되면 `$24.99 ~ $29.99`가 설득력 있다.

1. 3시즌 이상 장기 세이브 루프 검증
2. 경제/계약/이적 협상 텐션 강화
3. DayView 중심 UX 압축과 정보 구조 정리
4. 뉴스/서사의 기억성 강화
5. 릴리스 체크리스트 기준 수동 QA 반복 완료

### 아직 비추천하는 가격대

- `$34.99+`
  - 이유: 현재는 FM26/F1 Manager/WE ARE FOOTBALL과 직접 비교당하는 구간인데, 브랜드/라이선스/마감도에서 방어가 어렵다.

## 8. Recommended Next Moves

### Critical

- EA 전에는 기능 추가보다 `반복 플레이 응집력`과 `정보 구조 압축`에 집중
- 가격 문구는 “LoL esports management sim”으로 잡고 FM 직격 비교는 피함

### High

- 경제/협상/이적 루프를 더 아프게 만들어 의사결정 무게를 올릴 것
- 뉴스/프랜차이즈 서사를 “사람 기억” 중심으로 재구성할 것

### Medium

- 스토어 페이지용 비교 문구와 USP를 별도 정리
- 장기 세이브 리포트 문서와 플레이테스트 일지 누적
