## Short Description

LoL e스포츠 팀을 운영하며 패치 메타, 스크림, 밴픽, 코치 브리핑, 선수단 분위기를 읽어 시즌을 살아남는 매니지먼트 게임.

## Long Description

축구 FM의 표면만 가져온 게임이 아니라, LoL e스포츠 운영의 리듬을 중심에 둔 매니지먼트 경험입니다.

패치가 바뀌면 메타 해석이 흔들리고, 스크림 결과는 코치 브리핑과 밴픽 우선순위로 이어집니다. 상대 리포트와 선수단 분위기, 보드 압박, 샐러리캡과 계약 협상까지 연결되며 한 경기 결과가 시즌 전체 흐름에 후폭풍을 남깁니다.

EA 버전에서는 핵심 매니저 루프와 시즌 운영, 경기 준비, 라이브 매치 연출, 뉴스/서사, 오프시즌 전환 안정성에 집중합니다. 이후 업데이트에서는 장기 커리어 변주, 더 깊은 국제전 콘텐츠, 반복 플레이용 서사 이벤트, 추가 밸런스 튜닝을 확장합니다.

## Key Features

- 패치 메타, 스크림, 코치 브리핑, 상대 리포트를 하나의 운영 브리프로 확인
- 밴픽과 전술 준비가 실제 경기 해설과 결과 해석으로 이어지는 LoL 방송 감성
- 선수 만족도, 역할 기대, 감독 정체성, 스태프 궁합, 보드 압박이 연결되는 시즌 운영
- 사건 체인형 뉴스와 라이벌리, 국제전, 세대교체, 장기 커리어 서사
- 샐러리캡, 예산 압박, 실패 협상 비용까지 반영되는 팀 재정 관리
