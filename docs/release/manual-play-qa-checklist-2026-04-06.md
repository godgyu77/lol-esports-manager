# 수동 플레이 QA 체크리스트

## 목적

이 문서는 실제 플레이를 진행하면서 빠르게 이상 여부를 확인하기 위한 수동 QA 체크리스트다.  
긴 리포트용이 아니라, "한 번 플레이할 때 무엇을 체크해야 하는가"에 집중한다.

## 사용 방법

- 한 세션당 30분~60분 정도만 플레이해도 된다.
- 모든 항목을 한 번에 다 보지 않아도 된다.
- `정상 / 어색함 / 버그` 중 하나로 표시하면 된다.
- 이상이 있으면 가능하면 어떤 화면에서, 어떤 행동 직후 발생했는지 같이 적는다.

## 1. 첫인상 / 진입 흐름

- 메인 메뉴에서 무엇을 눌러야 하는지 바로 이해된다.
- 새 게임 시작 흐름이 자연스럽다.
- 감독 생성/팀 선택/시즌 목표 설정 흐름이 끊기지 않는다.
- 첫 진입 화면에서 영어, 깨진 한글, 의미 불명 문구가 보이지 않는다.
- 버튼 우선순위가 명확하다.
- "지금 무엇을 해야 하는지" 헷갈리지 않는다.

## 2. DayView 코어 루프

- DayView에 들어갔을 때 가장 중요한 일이 바로 보인다.
- 우선순위 카드가 실제 위험도와 잘 맞는다.
- 세팅 미완료, 재정 압박, 후폭풍, 경기 준비 중 무엇이 더 급한지 납득된다.
- 다음 행동 버튼이 자연스럽고, 눌렀을 때 기대한 화면으로 이동한다.
- 하루를 넘기기 전에 "이걸 확인하고 가야겠다"는 정보가 보인다.
- 불필요하게 정보가 중복되거나 산만하지 않다.

메모:

## 3. 재정 / 운영 압박

- 재정 화면에서 현재 팀 상태를 빠르게 이해할 수 있다.
- 예산 활주로, 주간 소모, 실패 협상 비용이 체감된다.
- 돈이 부족할 때 실제로 압박이 느껴진다.
- 무리한 협상이나 반복 접촉이 손해처럼 느껴진다.
- 보드 반응과 재정 상태의 연결이 납득된다.
- 숫자는 많아도 "그래서 지금 위험한가"가 명확하다.

메모:

## 4. 훈련 / 전술 / 경기 준비

- 훈련 화면에서 이번 주에 무엇을 조정해야 하는지 이해된다.
- 스크림/멘토링/개별 훈련이 왜 중요한지 보인다.
- 전술 화면에서 바꾸는 값이 어떤 효과를 내는지 이해된다.
- 경기 전 준비 흐름이 귀찮기만 하지 않고 의미 있게 느껴진다.
- 추천 메시지와 실제 선택지가 연결된다.
- 과도하게 많은 정보 때문에 결정을 미루게 되지 않는다.

메모:

## 5. 드래프트 / 경기 진행

- 드래프트 화면에서 현재 턴과 해야 할 행동이 명확하다.
- 블루/레드 차례, 밴/픽 상태를 헷갈리지 않는다.
- 경기 화면에서 지금 누가 유리한지 읽힌다.
- 중계 HUD, 미니맵, 팀 상태판의 정보 위계가 괜찮다.
- 경기 속도 조절, 다음 단계 진행, 세트 전환이 자연스럽다.
- 경기 중 텍스트가 너무 많거나 너무 적지 않다.

메모:

## 6. 뉴스 / 서사 / 커리어 체감

- 뉴스가 단순 로그가 아니라 "우리 팀 이야기"처럼 느껴진다.
- 최근 결과, 재정 상태, 라이벌전, 감독 커리어가 어느 정도 연결돼 보인다.
- 시즌을 이어갈수록 기억에 남는 사건이 생긴다.
- 커리어 화면이 장기 플레이 동기를 준다.
- "기사 수는 많지만 기억나는 게 없음" 같은 느낌이 들지 않는다.

메모:

## 7. UI 완성도 / 사용감

- 텍스트가 잘 읽힌다.
- 색상, 강조, 경고 메시지 구분이 명확하다.
- 클릭 가능한 요소와 정보 요소가 구분된다.
- 스크롤이 과도하게 길거나 피곤하지 않다.
- 비슷한 버튼이 여러 개 있어서 헷갈리지 않는다.
- 화면마다 말투와 용어가 크게 흔들리지 않는다.

메모:

## 8. 버그 / 안정성

- 화면 전환 중 깨짐, 겹침, 비정상 레이아웃이 없다.
- 버튼을 눌렀는데 반응이 없거나 이상한 화면으로 가지 않는다.
- 저장/불러오기가 불안하지 않다.
- 경기 전후, 다음 날 진행, 시즌 종료 같은 큰 전환에서 오류가 없다.
- 수치가 비정상적으로 튀거나 말이 안 되는 결과가 보이지 않는다.
- 플레이를 막는 치명적 문제 없이 한 사이클을 끝낼 수 있다.

## 우선 보고 기준

### 바로 수정 우선

- 진행 불가
- 저장/불러오기 문제
- 잘못된 화면 이동
- 심한 한글 깨짐
- 경기/시즌 진행이 망가지는 밸런스 붕괴

### 빠른 개선 우선

- 다음 행동이 불분명함
- 정보 위계가 약해서 중요한 걸 놓침
- 재정 압박/보드 압박이 숫자만 있고 감정이 안 옴
- 뉴스/서사가 많은데 기억에 남지 않음
- 버튼/카드/패널이 많아 피로함

## 추천 최소 플레이 시나리오

### 시나리오 A. 첫 진입

- 새 게임 시작
- 감독 생성
- 팀 선택
- 시즌 목표 설정
- 첫 DayView 진입

### 시나리오 B. 운영 루프

- DayView 확인
- 재정 화면 확인
- 훈련/전술 조정
- 하루 진행 3~5회 반복

### 시나리오 C. 경기 루프

- 프리매치 진입
- 드래프트 진행
- 라이브 경기 관전
- 경기 후 결과 확인
- 다음 일정까지 진행

### 시나리오 D. 장기 체감

- 여러 날/여러 주 진행
- 뉴스/커리어/보드/재정 변화를 같이 확인
- "계속 플레이하고 싶은가"를 마지막에 체크
