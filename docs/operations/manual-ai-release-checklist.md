# AI 수동 검수 체크리스트

## 설정 화면
- `Settings > AI 설정`에서 `템플릿`, `Ollama 로컬`, `OpenAI/Claude/Gemini/Grok` provider 카드가 정상적으로 보이는지 확인
- provider 카드에 API 키 저장 여부가 올바르게 표시되는지 확인
- 선택한 provider 아래 `현재 실행 정책` 설명이 바뀌는지 확인
- `기능별 AI 정책` 카드가 5개 모두 보이는지 확인

## 연결 테스트
- 클라우드 provider 선택 후 API 키 저장이 되는지 확인
- `연결 테스트` 버튼이 성공/실패 메시지를 정확히 보여주는지 확인
- 템플릿 모드에서는 연결 테스트가 필요하지 않다는 안내가 보이는지 확인
- Ollama가 꺼진 상태에서는 로컬 연결 실패 후 폴백 경로가 유지되는지 확인

## 실제 게임 흐름
- 드래프트 조언 호출 시 AI 실패가 발생해도 템플릿 조언이 즉시 보이는지 확인
- 라이브 매치 중 중계 문구와 채팅이 AI 실패 시에도 계속 출력되는지 확인
- 뉴스/팬 반응 생성이 AI 실패 시에도 템플릿 콘텐츠로 이어지는지 확인
- 면담/스카우팅 기능이 AI 없이도 결과를 반환하는지 확인

## 최종 안정성
- AI를 완전히 끈 상태에서도 경기 진행, 뉴스 소비, 면담 흐름이 끊기지 않는지 확인
- 저사양 환경에서 템플릿 모드로 플레이했을 때 기능 누락 없이 진행 가능한지 확인
- 고사양 환경에서 Ollama 사용 시 체감 지연이 과도하지 않은지 확인
